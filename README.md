# Swirlt
Simple fullstack FOSS self-hostable app to share text, media, URLs very quickly via web or CLI

## Stack
- Backend: Java, Spring Boot, Gradle
- Frontend: TypeScript, Angular, PNPM
- Deploy: Bash, Docker, Kubernetes
- Database: SQL, PostgreSQL
 
## Deploy
App can be deployed with dev.sh or prod.sh for different environments (Docker must be installed), default port for backend is 8080 and for frontend is 80 for prod and 4200 for dev
